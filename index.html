About

This Figma Library was built by the community for the community.

After seeing many requests for a Figma version of Relume Library (the world’s largest Webflow component library), the team at Minimal Square took it upon themselves to build it. Not only have they done an epic job of building 900+ pixel-perfect Figma components, they were generous enough to give this out for FREE.

Why did we build this?

To improve your Figma to Webflow workflow

  • Design wireframes for any project at speed.
  • Ensure that designs are compatible with Relume Library Webflow components.
  • Make design changes to components faster.

    • To collaborate better with clients & team members

  • Design wireframes for clients in minutes.
  • Leverage the power of Figma’s collaboration tools and receive feedback on designs before you begin building.
  • Handover designs to Webflow developers that are compatible with Relume Library.

    • You can use this Figma Kit in unlimited projects. Like the Relume Library, these components are designed to be as neutral, flexible, and scalable as possible to use as a kick-starter for any project.

    We’d love for you to try it out for your next project.

    How to use it with Relume Library and 10x your Webflow build speed

    1. Copy the name of the Figma component.

    2. Paste it into the Relume Library search bar to find the matching Webflow component.

    3. Hit the copy button and copy the Webflow component to your clipboard.

    4. Lastly, paste it into your Webflow project (Cmd + V or Ctrl + V)
